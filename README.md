# Smart Expense Tracker

A clean and efficient Python-based Expense Tracker developed to manage daily expenses, monitor monthly budgets, and analyze spending patterns.

The application stores expense records permanently using file handling and provides visual expense analysis using matplotlib.

## Features

- Add and manage expenses
- Automatic date generation
- Category-wise expense tracking
- Monthly budget analysis
- Expense limit alerts
- Permanent data storage
- Highest expense tracking
- Expense summary reports
- Graphical visualization using matplotlib
- Exception handling for invalid inputs

## Technologies Used

- Python
- File Handling
- Exception Handling
- Datetime Module
- Matplotlib
